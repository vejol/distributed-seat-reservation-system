# Server for Distributed Seat Reservation System

This project includes a Flask server that provides RESTful APIs for the Distributed Seat Reservation System UI.

## Getting Started

1. Install virtual environment and dependencies:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
2. Run the Flask server:
   ```bash
   python -m server
   ```

The server will start on `http://localhost:5000`.
